 3. Typography System (VERY IMPORTANT)

From the site style:

Clean sans-serif
Strong hierarchy
High readability

 We replicate that.

 Primary Font (UI & Body)
 Inter (Recommended)
font-family: 'Inter', sans-serif;

Why:

Clean
Modern
Perfect for dashboards
Matches SaaS feel
 Secondary Font (Optional Headings)
 Poppins
font-family: 'Poppins', sans-serif;

Why:

Slightly more personality
Still clean and professional
 Typography Hierarchy
Headings
Level	Size	Weight
H1	32–40px	Bold
H2	24–32px	Semi-bold
H3	18–24px	Medium
Body
Type	Size
Normal text	14–16px
Small text	12–13px
UI Elements
Element	Style
Buttons	Medium / Semi-bold
Labels	Medium
Inputs	Regular
 Rule (VERY IMPORTANT)

Consistency in typography builds trust.

 Never mix random fonts
 Stick to max 2 fonts

 4. Spacing & Layout System

DFT should follow clean SaaS spacing:

Spacing Scale
4px → micro spacing
8px → small
16px → standard
24px → section spacing
32px → major blocks
Layout Feel
Lots of whitespace
Card-based UI
Soft shadows
Rounded corners (rounded-2xl)
