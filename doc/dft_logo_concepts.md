# 🕊️ Dovepeak-Fintrack (DFT) Logo Concepts

I have designed three variations of a modern, sleek vector logo for the **Dovepeak-Fintrack** application. These SVGs have been directly saved to your project's `public` folder so you can start using them immediately!

## 1. Full Logo (Combination Mark)

![Dovepeak Full Logo](file:///c:/Users/josep/dovepeak-fintrack/public/logo.svg)

**File location:** [public/logo.svg](file:///c:/Users/josep/dovepeak-fintrack/public/logo.svg)

**Design Breakdown:**
- **The Icon:** Features a two-part geometric form. The sharp, upward trajectory creates a striking "peak" or "chart rising" effect signaling financial growth. Simultaneously, the two elements are angled to represent a soaring dove in flight, balancing the aggression of pure financial aesthetics with the calm, trusting presence of a dove.
- **The Accent:** A perfectly balanced soft blue (`#3B82F6`) dot sits atop the peak like an ascending sun or an eye of the bird, adding a subtle splash of tech-forward accent color.
- **Typography:** "Dovepeak" uses a bold `800` weight to show stability and trust, while "Fintrack" is lighter (`400` weight) in the primary Emerald green to provide visual contrast and indicate the application's function.

### 🌓 Auto Dark-Mode Support
All SVG files have been styled internally with CSS `@media (prefers-color-scheme: dark)` queries. 
This means if you drop them directly into an `<img>` tag or CSS `background-image`, they will instantly and perfectly adapt their slate text (`#0F172A`) and monochrome colors into off-white (`#F8FAFC`) when a user has Dark Mode active on your application!
